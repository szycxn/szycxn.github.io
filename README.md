## 3. 作业3

我小时候总是把哈利·波特想象成一个勇敢、精明、坚韧的人。因此，我很好奇书中是如何描述他的，是否和我的印象相符。于是，我决定数一数第一本哈利·波特书中形容词出现的次数，以及它们与哈利一起出现的次数，并创建一个词云图来总结这些发现。

具体而言，它实现了以下步骤：

读取指定文本文件并将其分段。

对每个段落进行分词和词性标注，提取出形容词。

统计每个形容词与哈利同行出现的次数。

根据统计结果生成词云图，其中形容词的大小表示其出现的频率。
